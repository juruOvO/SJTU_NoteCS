## Thread Concept
---
### Defination
- A THREAD is a basic unit of CPU utilization
	- A thread comprises
		- a thread ID
		- a program counter
		- a register set
		- a stack
	- What a thread shares with other threads to a same process
		- code section
		- data section
		- other OS resources, such as open files
