## Hall's Theorem
$$
\begin{align}
\text{In bipartite set \(G\), \(G\) contains a matching of size \(\mid A\mid\) iff }\\
\mid N(S)\mid\geq \mid S\mid~(\forall S\subseteq A)\\
\text{N(S) means the neighbours of set S.}
\end{align}
$$
